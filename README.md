# light-dark-switch
 
light-dark-switch使用lit构建。

# Quick Start

## Install

```
npm i light-dark-switch
```

## Import

在index.html中引入

```html
<script type="module" src='switch/index.js'></script>
```

或者在main.js中引入
```js
import 'switch/index.js'
```

## Usage

```html
<script type="module" src='switch/index.js'></script>

<light-dark-switch md sync selected></light-dark-switch>
```
