# light-dark-switch
 
light-dark-switch使用lit构建。

## Quick Start

### Install

### Import

在index.html中引入

```html
<script type="module" src='switch/es/src/light-dark-switch.mjs'></script>
```

或者在main.js中引入
```js
import 'switch/es/src/light-dark-switch.mjs'
```

### Usage

```html
<script type="module" src='switch/es/src/light-dark-switch.mjs'></script>

<light-dark-switch md sync selected></light-dark-switch>
```
